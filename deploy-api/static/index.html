Quick Start

1. Install the npm package

npm install -g puo-memo-mcp

2. Configure Claude Desktop

Add to your Claude Desktop config:

{ "mcpServers": { "puo-memo": { "command": "npx", "args": ["puo-memo-mcp"], "env": { "PUO_MEMO_API_KEY": "your-api-key-here" } } } }

3. Start using memory commands

// Store a memory await memory("Important meeting notes from today...") // Recall memories await recall("meeting notes")
